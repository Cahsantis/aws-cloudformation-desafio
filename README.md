# Minha Primeira Stack com AWS CloudFormation

## Objetivo
O objetivo deste projeto é criar minha primeira Stack utilizando *AWS CloudFormation*, aplicando os conceitos aprendidos nas aulas da DIO e documentando todo o processo para aprendizado e futuras implementações.

## Etapas do Projeto

1. *Revisão das aulas*
   - Assisti a todas as vídeo-aulas do módulo de CloudFormation.
   - Anotei conceitos importantes sobre stacks, templates, parâmetros, recursos e outputs.

2. *Criação do Template*
   - Criei o arquivo template.yaml definindo os recursos da stack (Ex.: S3 Bucket).
   - Incluí parâmetros e outputs para facilitar o reuso.

3. *Deploy da Stack*
   - Acesse o console da AWS → CloudFormation.
   - Clique em "Create Stack → With new resources (standard)".
   - Enviei o arquivo template.yaml.
   - Configurei os parâmetros solicitados.
   - Criei a stack e verifiquei se todos os recursos foram provisionados corretamente.
